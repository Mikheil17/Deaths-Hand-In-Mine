# Death's Hand In Mine
Experience the passage of time as you go through each day, growing older and coming to realize the fleeting nature of life.

## What It Is
A Unity VR game using Inverse Kinematics for full embodiment.

## Key Features
- Realistic IK body movement
- Fully interactible environments
- Storyline exploring the passage of time, the cycle of life, and acceptance of mortality

## Tech
Unity | C#

## Context
Created as part of a VXR class, focusing on VR embodiment mechanics.
